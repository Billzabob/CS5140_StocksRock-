# Data Analysis of Stocks

Using regression on high dimensional data points representing stock days to predict the change in price 
at certain increments (2 days, 4 days, 8 days, etc)

# Features

- Automatically downloads data from Google finance, and Google trends and extracts useful features from the data
- The preprocessing is seperate from the analysis so you can preprocess for one stock / company and try different 
analysis techniques
- Finds the most important features, used for both reducing features and seeing what affects the price change the most
